# N688TW 環球飛行官網 - 設計規劃

## 選定設計方向：極簡航空未來主義

### 設計運動 (Design Movement)
**極簡航空未來主義 (Minimalist Aerospace Futurism)**
融合航空工業的精密感、未來科技的冷峻美學，以及極簡主義的克制優雅。

### 核心原則 (Core Principles)
1. **深度黑色基調**：純黑背景（#000000）象徵太空、夜間飛行、專業沉著
2. **極簡幾何**：使用直線、圓形、網格構成航空儀表盤的視覺語言
3. **高對比亮點**：銀白色、霓虹藍、金色點綴，模擬飛行儀表的發光效果
4. **動態流線**：微妙的動畫和過渡效果，暗示飛行、速度與運動

### 色彩哲學 (Color Philosophy)
- **主色**：純黑 (#000000) - 深邃、專業、現代
- **輔助色**：
  - 銀白 (#E8E8E8) - 高對比、易讀性、科技感
  - 霓虹藍 (#00D9FF) - 飛行儀表的發光感、能量、現代科技
  - 金色 (#D4AF37) - 高級感、獎項、成就感
  - 深灰 (#1A1A1A) - 卡片背景、層次感

### 版面範式 (Layout Paradigm)
- **非對稱網格**：避免居中單調，採用左右不對稱的內容分佈
- **分層設計**：使用深淺黑色創造視覺深度
- **流動路徑**：地圖、內容流向從左到右、上到下，暗示飛行軌跡

### 標誌性元素 (Signature Elements)
1. **飛行軌跡線**：連接各個航點的流暢曲線，象徵環球飛行路線
2. **儀表盤圓形**：圓形進度指示、數據展示，模擬飛行儀表
3. **網格背景**：微妙的網格紋理，增加科技感與深度

### 互動哲學 (Interaction Philosophy)
- **懸停效果**：卡片、按鈕懸停時發光、浮起，模擬按鈕被激活的感覺
- **平滑過渡**：所有狀態變化使用流暢的 easing，避免生硬跳躍
- **微互動**：滑鼠移入時細微的光暈、陰影變化

### 動畫指南 (Animation Guidelines)
- **地圖動畫**：飛行路線逐點顯示，每個航點閃爍發光
- **卡片進場**：從下方淡入並向上滑動
- **懸停發光**：按鈕/卡片懸停時，邊框或背景發出藍色光暈
- **頁面轉換**：淡入淡出效果，保持流暢感

### 排版系統 (Typography System)
- **標題字體**：Orbitron / Space Mono（科技感、幾何線條）
- **正文字體**：Inter / Roboto（清晰易讀、現代）
- **層級結構**：
  - H1: 48px, 700 weight, 字母間距 2px
  - H2: 32px, 600 weight, 字母間距 1px
  - Body: 16px, 400 weight, 行高 1.6

## 實現細節

### 頁面結構
1. **首頁**：互動世界地圖 + 飛行信息卡片
2. **贊助商頁**：Logo 網格展示
3. **直播頁**：YouTube 嵌入 + 實時信息
4. **故事頁**：林睿哲的故事，分章節展示

### 視覺資產
- 生成高質量的航空相關背景圖片
- 飛行軌跡地圖（自繪或使用 Google Maps）
- 飛機型號圖片（SLING 4 TSi）

### 技術實現
- React 19 + TypeScript
- Tailwind CSS 4（深色主題）
- Framer Motion（動畫）
- Google Maps API（互動地圖）
- Lucide React（圖標）
